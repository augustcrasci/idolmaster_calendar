==================== 아이마스 통합 캘린더 ====================

아이마스 공식 포털의 스케줄을 포함해
- 공식 일정 / 이벤트
- 아이돌 / 캐스트 생일
- 티켓 응모 마감
- 한국 / 일본 공휴일

데이터를 자동으로 수집하고,
월간 / 주간 캘린더 형태로 한눈에 볼 수 있게 만든 프로젝트입니다.

================================================================

[필수 사항]
- Python 3.x
- 인터넷 연결

[설치]
아래 명령은 CMD 또는 PowerShell에서 실행하면 됩니다.

1. 필요한 패키지 설치
pip install playwright requests beautifulsoup4 holidays

2. Playwright 브라우저 설치
playwright install

추가로 필요한 pip 패키지는 없습니다.
- `launcher.pyw`의 작은 실행 창은 Python 기본 포함 라이브러리인 `tkinter`를 사용합니다.
- 일반적인 Windows Python 설치에서는 별도 설치가 필요 없습니다.

================================================================

[실행 방법]

가장 편한 방법:
- `launcher.pyw` 더블클릭
- 콘솔창 없이 작은 실행 창이 열립니다.
- 여기서 아래 3가지 중 하나를 버튼으로 선택할 수 있습니다.
  - DB 업데이트만
  - 뷰어만 실행
  - DB 업데이트 후 뷰어 실행

가장 추천하는 사용 방식:
- 처음 실행할 때는 `업데이트 후 뷰어 실행`
- 이후 화면만 볼 때는 `뷰어만 실행`

================================================================

[폴더 구조]

launcher.pyw
- 메인 실행기
- 더블클릭용 GUI 실행기
- 콘솔 없이 버튼으로 실행 가능

app/
- 파이썬 로직 모음
- 공식 일정 크롤러
- 생일 수집기
- 티켓 마감 수집기
- 공휴일 생성기
- 로컬 서버 / 뷰어

data/
- 수집된 JSON 데이터 저장 폴더
- schedule_data.json
- birthday_data.json
- ticket_grouped_data.json
- holidays.json

web/
- 캘린더 화면 파일
- index.html

================================================================

[데이터 설명]

schedule_data.json
- 아이마스 공식 포털 일정 데이터

birthday_data.json
- 아이돌 / 캐스트 생일 데이터

ticket_grouped_data.json
- 라이브 / 이벤트 티켓 응모 마감 데이터

holidays.json
- 한국 / 일본 공휴일 데이터

================================================================

[주의 사항]
- 공식 사이트 구조가 바뀌면 크롤러가 일시적으로 동작하지 않을 수 있습니다.
- 티켓 데이터는 "시작일"이 아니라 "마감일" 기준으로 표시됩니다.
- 진행 중 이벤트는 포털 표시 방식에 따라 날짜가 보정되어 저장됩니다.
- 메모 / 테마 설정은 브라우저 로컬 저장소(localStorage)에 저장됩니다.
- 브라우저 화면이 이상하게 보이면 강력 새로고침(Ctrl+F5)을 해 주세요.
- 뷰어는 로컬 서버를 사용하므로, 새로고침을 정상적으로 쓰려면 백그라운드 서버가 살아 있어야 합니다.
